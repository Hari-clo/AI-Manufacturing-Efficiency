AI-Based Manufacturing Efficiency Intelligence System

Overview
This project presents an AI-driven system for real-time classification of manufacturing efficiency using sensor and network data.

## Key Features
- Real-time efficiency classification (High / Medium / Low)
- Confidence scoring
- Sensor and network impact visualization
- Actionable insights and recommendations

## Technologies Used
- Python
- Scikit-learn
- Streamlit
- SHAP (for explainability)

## Insights
- Sensor instability significantly impacts efficiency
- Network latency affects production coordination
- Energy inefficiency reduces output performance

## Recommendations
- Improve machine calibration
- Optimize network infrastructure
- Monitor power consumption trends
